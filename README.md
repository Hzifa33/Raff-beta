# رَفّ 4.0.0

**نظام مكتبات عربي محلي متكامل يعمل دون اتصال بالإنترنت.**

رَفّ تطبيق سطح مكتب لنظام Windows يضع الفهرسة، والمقتنيات، والنسخ المادية، والمستعيرين، والإعارة، والحجوزات، والجرد، والفروع، والتزويد، والدوريات، والتقارير، وسلامة البيانات داخل واجهة عربية RTL منظمة حسب مهمة المستخدم.

> المبدأ الأساسي: الباحث يرى فهرسًا هادئًا وبسيطًا، بينما تظهر أدوات الإعارة والفهرسة والإدارة فقط للأدوار التي تحتاجها.

## ما الجديد في 4.0

- نموذج بيانات احترافي يفصل بين **السجل الببليوغرافي** و**المقتنى** و**النسخة المادية**.
- باركود مستقل لكل نسخة ولكل جزء مع تتبع الحالة والموقع.
- ملفات مستقلة للمستعيرين، وفئات عضوية، وسياسات إعارة قابلة للتخصيص.
- إعارة وإرجاع وتجديد وحجوزات وقوائم انتظار وإرجاع جزئي للكتب متعددة الأجزاء.
- فروع ومواقع ورفوف ونقل نسخ بين الفروع مع الاستلام والإلغاء وسجل تدقيق.
- جلسات جرد بالباركود تكشف المفقود، والمكرر، والمجهول، والموجود في مكان خاطئ.
- ضبط استنادي للمؤلفين والجهات والموضوعات ودور النشر والسلاسل.
- استيراد وتصدير محلي: MARCXML، MARC 21 ISO 2709، BibTeX، RIS، Dublin Core، JSON-LD، وحزمة تبادل رَفّ.
- OPAC محلي للبحث العام على الجهاز أو الشبكة المحلية دون خدمات خارجية.
- التزويد والمشتريات، والدوريات، ووضع دار النشر.
- مستخدمون محليون وأدوار وصلاحيات وPIN مشفر وسجل تدقيق.
- سلة محذوفات، ونسخ احتياطية، وفحص سلامة، وإصلاح آمن.
- تنبيهات محلية للإعارات والحجوزات والدوريات والتزويد ونقل النسخ.
- حقول مخصصة للسجلات والنسخ والمستعيرين وطلبات التزويد.
- ترقية بيانات 2.7 تلقائيًا مع إبقاء طبقة توافق للشاشات القديمة.

## مساحات العمل

| المساحة | الغرض |
|---|---|
| الباحث والاستكشاف | البحث، الإتاحة، مواقع الكتب، وقوائم القراءة فقط |
| الإعارة وخدمة القراء | المستعيرون، الإعارة، الإرجاع، التجديد، الحجوزات والجرد |
| الفهرسة والمجموعات | السجلات، المقتنيات، النسخ، الضبط الاستنادي والتبادل |
| الإدارة والتقارير | الفروع، السياسات، المستخدمون، التقارير، السجل والنسخ |
| كل الأدوات | مساحة المدير وأمين المكتبة |

الأدوار غير الإدارية لا تستطيع اختيار مساحة تعرض أدوات خارج نطاق دورها، كما تُحجب مجموعات البيانات الحساسة عن واجهة الباحث.

## العمل دون إنترنت

- لا حساب سحابي.
- لا تتبع ولا تحليلات استخدام.
- لا ذكاء اصطناعي ولا جلب بيانات من خدمات خارجية.
- لا خطوط أو أيقونات من CDN.
- كل البيانات والباركود والتقارير والتبادل تتم محليًا.
- OPAC المحلي يستخدم `localhost` أو الشبكة المحلية فقط عند تفعيله صراحة.

## متطلبات التشغيل

- Windows 10 أو Windows 11، بنواة 64-bit.
- Node.js لبناء المصدر فقط.
- قارئ باركود USB وطابعة ملصقات اختياريان.

## تشغيل المصدر


## حدود النسخة

هذه الحزمة تنفذ النواة المحلية المتقدمة. تكاملات الأجهزة أو الأنظمة المؤسسية التي تتطلب جهازًا فعليًا أو بروتوكول مورد، مثل RFID وSIP2 وNCIP، ليست معتمدة ميدانيًا داخل هذه الحزمة. صيغ MARC المدعومة تغطي الحقول الأساسية المستخدمة في رَفّ، وليست محرر MARC شاملًا لكل حقول المعيار.

## الخصوصية والأمان

- `contextIsolation` و`sandbox` مفعّلان.
- `nodeIntegration` معطل داخل الواجهة.
- واجهة `preload` محدودة.
- سياسة CSP محلية.
- رموز PIN تخزن باستخدام `scrypt` ولا ترسل إلى الواجهة.
- البيانات الشخصية لا تظهر في لقطة واجهة دور الباحث.
- الروابط الخارجية مقيدة بقائمة سماح.
