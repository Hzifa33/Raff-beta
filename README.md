## عن رَفٌّ

**رَفٌّ** برنامج مكتبي عربي لإدارة وفهرسة المكتبات، صُمم للمكتبات الصغيرة، المدارس، المراكز التعليمية، المساجد، المبادرات الخيرية، والباحثين الذين يريدون نظامًا بسيطًا وسريعًا بدل الجداول المبعثرة والبحث اليدوي.

يعمل البرنامج **محليًا وبلا إنترنت**، بواجهة عربية كاملة (RTL)، وخطوط مدمجة، وبلا أي إعلانات أو تتبع.

## أبرز مميزات الإصدار 2.4.0

### الواجهة وتجربة الاستخدام
- تصميم **Minimalist زجاجي بالكامل** مستوحى من تطبيقات سطح المكتب الاحترافية، دون خلفيات مصوّرة أو ألوان مرهقة.
- وضعان **نهاري وليلي** بأبيض هادئ وأسود غير ناصع، مع حفظ الوضع المختار تلقائيًا.
- قائمة جانبية مرنة **تُطوى وتُفتح مثل VS Code**، مع تلميحات للأيقونات وحفظ حالتها بين مرات التشغيل.
- انتقالات ناعمة بين الصفحات، وحركات دقيقة للأزرار والبطاقات والنوافذ دون التأثير في سرعة البرنامج.
- اختصارات جديدة: `Ctrl+B` لطي القائمة و`Ctrl+Shift+L` لتبديل المظهر، مع استمرار جميع الاختصارات السابقة.

### الفهرسة وبيانات الجرد
- تسجيل: اسم الكتاب، المؤلف، دار النشر، التصنيف، الطبعة، سنة النشر، والملاحظات.
- **حقول جرد موسّعة**: السلسلة، الترتيب داخل السلسلة، الكلمات المفتاحية، حالة النسخة (جيدة/مقبولة/تالفة/مفقودة)، جهة الاقتناء، والرف.
- **السعر وعدد الأجزاء** لكل كتاب (اختياريان)، يظهران في التصدير والتقارير.
- اقتراحات تلقائية أثناء الكتابة للمؤلف، دار النشر، التصنيف، السلسلة، والرف.

### البحث
- بحث فوري لا يتأثر بالتشكيل أو التطويل، ويفهم اختلاف الحروف: أ، إ، آ، ا، ى، ي، ة، ه.
- **البحث المتقدم** يغطي: العنوان، المؤلف، دار النشر، الرقم المرجعي، التصنيف، **السلسلة، الرف، الكلمات المفتاحية**، والمستعير.
- **السجل الكامل** جدول احترافي قابل للفرز بكل عمود (العنوان، المؤلف، السنة، السعر، الحالة، **تاريخ الإضافة**…)، مع **أعمدة قابلة لتغيير العرض بالسحب** (ونقرة مزدوجة على الفاصل تعيد العرض التلقائي)، وفلاتر: متاح، جزئي، معار، **متأخر**، و**يستحق قريبًا**، وتصفية بمدى سعري. يعرض النتائج على دفعات لأداء ثابت مهما كبرت المكتبة.

### الأرقام المرجعية والباركود
- إنشاء رقم مرجعي تلقائي لكل كتاب لتسهيل ترتيبه على الرف.
- **تعديل الرقم المرجعي** بعد الإضافة ومن نافذة التفاصيل، مع **حماية من التكرار** عند التعديل والاستيراد.
- **توليد باركود (Code 128)** من الرقم المرجعي لكل كتاب، يظهر فور الإضافة وفي صفحة المسح.
- **قارئ الباركود** يعمل تلقائياً من أي شاشة (قارئ USB يعمل كلوحة مفاتيح): امسح ملصق الكتاب فتظهر **صفحة بيانات الكتاب كاملة** فوراً (العنوان، المؤلف، الرقم، السعر، الرف، الأجزاء، الإتاحة، والإعارات المفتوحة).
- **طباعة الملصقات أو حفظها PDF**: ملصق مفرد لكتاب، أو ورقة ملصقات لكل المكتبة، أو حسب رف/مجال/سلسلة، أو **حسب نطاق رقمي** (مثلاً من raf-0001 إلى raf-0100) — مع خيار **الطباعة مباشرة** أو **الحفظ كملف PDF**.
- **ملصقات احترافية بهوية المؤسسة**: أضف **شعار واسم المكتبة/الأكاديمية** من الإعدادات فيظهران بوضوح على كل ملصق، مع **إطار أنيق**، وحجم ملصق مضبوط بدقة على ورق **A4** (الحجم الصغير مثالي للصقه خلف الكتاب دون حجب بياناته)، وعدد أعمدة قابل للضبط (3/4/5)، ومعلومات دقيقة شبه خفية (microtext) عن الكتاب.

### نظام الإعارة
- إعارة **نسخة كاملة** أو **جزء واحد أو عدة أجزاء معًا** للكتب متعددة الأجزاء، وتُحسب الإتاحة لكل جزء على حدة.
- **إرجاع جزء محدد من الإعارة** دون إغلاقها بالكامل: يمكن تسجيل عودة جزء واحد أو عدة أجزاء، مع بقاء الأجزاء الأخرى باسم المستعير وإتاحتها محسوبة بصورة مستقلة.
- **موعد إرجاع تلقائي بعد 30 يومًا**، قابل للتعديل عند الإعارة.
- **وسيلة تواصل** للمستعير و**ملاحظة قصيرة** لكل إعارة.
- تنبيهات «متأخر» و«يستحق خلال N يوم»، و**تصدير CSV للإعارات المتأخرة فقط**.

### لوحة المعلومات والتقارير
- لوحة معلومات ذكية: الإعارات المتأخرة، المستحقة خلال 7 أيام، نسبة اكتمال بيانات الجرد، قيمة الكتب المسعّرة، أعلى كتاب سعرًا، وعدد العناوين متعددة الأجزاء.
- استدعاءات قابلة للفرز حسب أي عمود، عبر ثمانية أبعاد: المستعيرون، دور النشر، المؤلفون، المجالات، سنوات النشر، **الرفوف، السلاسل، وحالة النسخ**.

### التخزين والأمان
- تخزين داخلي **أخف وأسرع**، مع بقاء تصدير JSON منسّقًا ومقروءًا، وكتابة آمنة لا تُتلف الملف عند انقطاعها.
- **نسخ أمان تلقائية** قبل الاستيراد وقبل حذف كل البيانات، و**زر نسخة فورية** (يُحتفظ بأحدث 20 نسخة).
- **فحص سلامة البيانات**: يكشف المتأخرات، الحقول الناقصة، الأرقام المكررة، والعناوين المكررة المحتملة.
- **زر فتح مجلد بيانات البرنامج** مباشرةً.

### التصدير
- تصدير المكتبة إلى **JSON، CSV، TXT، وPDF**، بالإضافة إلى CSV للمتأخرات.

### الخصوصية
- يعمل محليًا ولا يرسل بيانات مكتبتك إلى أي خادم خارجي، وبلا تتبع أو إعلانات.

## التحميل

حمّل آخر إصدار من صفحة الإصدارات الرسمية:

[GitHub Releases](https://github.com/Hzifa33/raff/releases/latest)

بعد التحميل:

```text
1. شغّل ملف التثبيت أو فك ضغط النسخة المحمولة إن وُجدت.
2. افتح برنامج رَفٌّ.
3. أضف أول كتاب أو استورد نسخة احتياطية.
```

## التحقق من سلامة الملف

استخدم قيمة **SHA256** التالية للتحقق من أن الملف الذي حمّلته هو نفس الملف المنشور رسميًا:

```text
ضع هنا قيمة SHA256 الناتجة من الأمر أدناه
```

للتحقق على **Windows PowerShell**، افتح PowerShell داخل مجلد التحميل ونفّذ:

```powershell
Get-FileHash -Algorithm SHA256 .\Raf_Setup_2.0.0.exe
```

أو للحصول على القيمة فقط دون باقي التفاصيل:

```powershell
(Get-FileHash -Algorithm SHA256 .\Raf_Setup_2.0.0.exe).Hash
```

إذا كان اسم الملف مختلفًا، استبدل `Raf_Setup_2.0.0.exe` باسم الملف الموجود لديك.

> ملاحظة: تُحسب قيمة SHA256 من ملف التثبيت النهائي بعد بنائه. انسخ ناتج الأمر أعلاه وضعه في الخانة المخصصة له.

## ملاحظة حول Windows SmartScreen

قد يعرض Windows رسالة مثل **Unknown Publisher** لأن البرنامج جديد وغير موقّع بشهادة Code Signing مدفوعة.

هذا لا يعني بالضرورة أن البرنامج ضار. لتقليل المخاطر:

```text
حمّل البرنامج فقط من GitHub Releases الرسمي
تحقق من SHA256 قبل التشغيل
لا تعتمد على نسخ مرسلة من مصادر مجهولة
```

## الخصوصية ومكان البيانات

رَفٌّ يعمل محليًا على جهازك.

لا يقوم البرنامج بـ:

```text
جمع بيانات المستخدم
رفع بيانات المكتبة إلى الإنترنت
تتبع الاستخدام
عرض الإعلانات
إرسال الكتب أو أسماء المؤلفين إلى خوادم خارجية
```

تُحفظ بيانات المكتبة في ملف محلي داخل مجلد بيانات التطبيق الخاص بالمستخدم، باسم:

```text
raff-library.json
```

وتُحفظ النسخ الاحتياطية بجانبه في مجلد:

```text
backups
```

يمكنك الوصول إلى المجلد مباشرةً من زر **«فتح مجلد بيانات البرنامج»** داخل الإعدادات.

## للمطورين

المشروع مبني باستخدام **Electron** و **JavaScript**، بمخزن بيانات محلي بصيغة JSON (بلا اعتماديات أصلية).

### تشغيل المشروع محليًا

```bash
npm install
npm start
```

### بناء نسخة Windows

```bash
npm run dist:win
```

سيتم إخراج ملفات البناء داخل مجلد:

```text
release
```

## رسالة المشروع

المعرفة تستحق نظامًا يحفظها، والكتاب الذي لا يُعثر عليه كأنه غير موجود.

**رَفٌّ** جاء ليجعل لكل كتاب مكانًا، ولكل باحث طريقًا أسرع إلى المعرفة.

## ثيم shadcn/ui — الإصدار 2.2.0

يستخدم التطبيق طبقة تصميم مستقلة في `src/css/shadcn-raff.css` تنقل مبادئ shadcn/ui إلى الواجهة الحالية دون تحويل المشروع إلى React أو تغيير منطق البرنامج. الثيم يحافظ على هوية «رَفّ» البنية والذهبية، ويعمل في الوضعين النهاري والليلي. توجد تفاصيل التحليل وخريطة المكوّنات في `docs/SHADCN_THEME_AUDIT.md`.
